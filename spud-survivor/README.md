# Spud Survivor

A first-person potato survivor dungeon game built with React, Vite, Three.js, and Tailwind CSS.

## Run Locally

Install dependencies:

```sh
pnpm install
```

Start the dev server:

```sh
pnpm run dev
```

Build for production:

```sh
pnpm run build
```

## Controls

- WASD: Move
- Mouse: Look
- Left click: Shoot
- Space: Dash
- Q: Special
- 1/2/3: Consumables

On touch devices, use the on-screen joystick, drag-to-look/fire area, and action buttons.
